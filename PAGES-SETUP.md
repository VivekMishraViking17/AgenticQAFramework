# Fix GitHub Pages deploy failure

## If workflow failed with "Get Pages site failed"

### Option A — One click in GitHub (recommended)

1. Open: https://github.com/VivekMishraViking17/AgenticQAFramework/settings/pages
2. **Build and deployment** → **Source** → select **GitHub Actions**
3. Go to **Actions** tab → **Deploy GitHub Pages** → **Re-run all jobs**

### Option B — Deploy from branch (no Actions needed)

1. Open: https://github.com/VivekMishraViking17/AgenticQAFramework/settings/pages
2. **Source** → **Deploy from a branch**
3. Branch: **main** · Folder: **/docs**
4. Save
