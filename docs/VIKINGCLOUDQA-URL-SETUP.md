## Step 1 — Create org (free, 2 minutes)

1. Open: https://github.com/organizations/plan?plan=free
2. Organization name: **`VikingCloudQA`**
3. Contact email: your `@vikingcloud.com` email
4. Choose **Free** plan → Create

---

## Step 2 — Transfer your repo (you own both — instant, no admin needed)

1. Open: https://github.com/VivekMishraViking17/AgenticQAFramework/transfer
2. New owner: **`VikingCloudQA`**
3. Confirm transfer

---

## Step 3 — Rename repo

1. Open: https://github.com/VikingCloudQA/AgenticQAFramework/settings
2. Repository name → **`agenticframework`**
3. Rename

---

## Step 4 — Enable GitHub Pages

1. Open: https://github.com/VikingCloudQA/agenticframework/settings/pages
2. Source: **Deploy from a branch**
3. Branch: **main** · Folder: **`/docs`**
4. Save

Wait 2–5 minutes.
